# Анализ результатов анкетирования «Студенческая жизнь»

Информационно-аналитическая система для анализа результатов анкетирования студентов.

## Технологии

- Python 3.12
- Streamlit — веб-интерфейс
- pandas — обработка данных
- plotly — интерактивные графики
- matplotlib — графики в PDF
- fpdf2 — генерация PDF
- openpyxl — работа с Excel
- pymorphy3 — лемматизация русского языка

## Установка

```bash
git clone <url>
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

## Функциональность

- Загрузка CSV/XLSX файлов (несколько файлов сразу)
- Предобработка данных (дубли, пропуски, типы)
- KPI-показатели
- Интерактивные графики (столбчатые, круговые, таблицы)
- Фильтрация по параметрам
- Аналитические выводы по каждому блоку
- Экспорт в PDF (с графиками) и XLSX
- Журнал событий

## Структура проекта

```
├── app.py              # Главное приложение
├── config.py           # Маппинг вопросов анкеты
├── requirements.txt    # Зависимости
├── data/               # Тестовые данные
├── modules/
│   ├── data_loader.py      # Загрузка данных
│   ├── preprocessor.py     # Предобработка
│   ├── analytics.py        # Статистика и KPI
│   ├── charts.py           # Визуализация
│   ├── filters.py          # Фильтрация
│   ├── conclusions.py      # Аналитические выводы
│   ├── excel_exporter.py   # Экспорт XLSX
│   ├── pdf_exporter.py     # Экспорт PDF
│   └── logger.py           # Журнал событий
```
